# ScanQR-WEB 操作说明书

## 一、系统简介

**ScanQR-WEB** 是一个微信小程序，用于制造车间的二维码扫码与流程卡管理系统。核心流程：管理员建卡 → 生成二维码 → 操作员扫码锁卡 → 填写工序数据 → 提交报工 → 全程可追溯。

### 角色体系

| 角色 | 权限范围 |
|------|----------|
| **系统管理员** (admin) | 全部功能：看板、卡片管理、日志、用户管理、角色管理、模板管理、工序字典 |
| **操作员** (operator) | 仅扫码上报、填报流程卡 |
| **质检员** (qc) | 扫码上报、卡片追溯 |

可自定义创建新角色并分配权限。

---

## 二、首次使用 — 系统初始化

1. 用微信开发者工具打开项目，编译运行
2. 进入小程序后会自动跳转到**登录页**
3. 点击底部的 **"初始化系统"** 按钮
4. 初始化完成后自动填充管理员账号和密码：
   - **账号**: `admin`
   - **密码**: `admin123`
5. 点击 **"登录"** 即可进入系统

> 初始化操作会创建：管理员账号、三个内置角色、示例模板（含常用工序列表）、测试卡片、数据字典。可重复执行（已存在的数据会跳过）。

---

## 三、登录

- 输入账号和密码，点击 **"登录"**
- 首次登录前必须先执行**系统初始化**
- 登录成功后，根据用户权限自动跳转：
  - 管理员 → **数据看板**
  - 操作员/质检员 → **扫码报到**

## 四、工序字典管理（后台维护工序列表）

建卡前需先维护可选工序列表，管理员在后台统一管理，操作员建卡时直接勾选，避免手输错误。

### 4.1 入口

**方法一**：**"我的"** 标签页 → **"工序与字典管理"**
**方法二**：在制卡片页 → 点击 **"建卡"** → 工序列表区域 → 点击 **"管理工序"**

### 4.2 管理工序列表

进入"数据字典"页面，找到 **"工序列表"** 字典：
- 点击 **"编辑"** → 在选项区域维护工序名称（每行一个）
- 可增删改任意工序

> 也可在该页面管理其他下拉选项（制程类型、不良原因等）。

---

## 五、在制卡片管理

在 **"在制"** 标签页，可管理所有流程卡。

### 5.1 状态筛选

- **加工中**：正在进行的卡片
- **已完工**：已标记完工的卡片
- **已作废**：已作废的卡片

### 5.2 建卡

点击 **"建卡"** 按钮：

1. **选择模板**：点击"请选择模板"，从列表中选取所需模板
2. **输入工单号**（如模板未自带）：填写工单号
3. **选择工序**：从工序列表中**勾选**所需的工序（可多选，按选择顺序排列）
   - 已选工序以蓝色标签展示，显示序号
   - 点击标签上的 ✕ 可移除
   - 如需补充列表中不存在的工序，点击 **"自定义工序"** 手动输入添加
4. **填写表头字段**：项目名称、交期等
5. 点击 **"创建"**

> 工序列表由管理员在"工序与字典管理"中维护，建卡时直接勾选即可，减少手动输入。

### 5.3 卡片操作

点击卡片的操作按钮，可执行：

| 操作 | 说明 | 权限要求 |
|------|------|----------|
| 生命周期追溯 | 查看卡片所有操作记录 | `card_trace` |
| 强制解锁 | 释放被他人占用的锁 | `card_unlock` |
| 标记完工 | 将卡片状态改为已完工 | 管理员 |
| 作废 | 将卡片状态改为已作废 | 管理员 |
| 恢复加工 | 将作废/已完工恢复为加工中 | 管理员 |

## 六、扫码报工（操作员日常操作）

### 6.1 获取工单号

进入 **"扫码"** 标签页，有 3 种方式获取工单号：

1. **扫描二维码**：点击扫码按钮，调用微信摄像头扫描二维码
2. **手动输入**：点击输入按钮，弹出对话框输入工单号
3. **上次工单**：自动记忆上次扫码的工单号

### 6.2 锁定流程卡

扫码成功后，系统会调用 `getAndLockCard` 锁定该流程卡：
- 锁定期限：**30 分钟**（超时后其他人可抢占）
- 同一张卡同时只能被一人锁定

### 6.3 填写流程卡

锁定成功后进入**流转卡填报页面**，包含表头字段、工序明细（每道工序含生产和品质部门）、入库信息等。

**快捷操作**：
- 点击 `datetime` 类型字段 → 自动填充当前时间
- 点击 `select` 类型字段 → 弹出下拉选择框

### 6.4 提交 / 放弃

- **提交**：保存所有填写数据并解锁卡片
- **放弃**：仅解锁卡片，不保存数据
- 关闭页面时未提交则**自动解锁**

## 十一、用户管理

**"我的"** → **"人员管理"**：

- **新增用户**：填写账号、密码、姓名、部门、角色、电话

## 十四、常见问题

**Q: 登录失败怎么办？**
A: 确保已执行系统初始化。若仍失败，检查账号是否被禁用，或联系管理员重置密码。

**Q: 扫码后提示"卡片已被锁定"？**
A: 该卡已被其他操作员锁定。等待提交后解锁，或联系管理员**强制解锁**。

**Q: 建卡时找不到需要的工序？**
A: 请管理员在"工序与字典管理"中编辑"工序列表"字典，添加所需工序。或建卡时点击"自定义工序"手动添加。

**Q: TabBar 看不到某些标签？**
A: TabBar 根据用户权限动态显示，无权限的标签不会出现。

**Q: 二维码可以重复扫描吗？**
A: 二维码含工单号，对应卡片未被锁定时可扫码填报。已完工/作废的卡片无法再锁定。
